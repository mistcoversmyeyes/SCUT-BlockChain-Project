# SCUT 区块链项目

基于区块链的智能合约开发项目，实现ERC20代币合约的前后端集成。

## 分支结构

### main
- **主分支**：包含项目的最新状态
- **当前状态**：整合了前端部署成果

### dev/back_end
- **后端开发分支**
- **完成内容**：实验2.1 - ERC20合约后端开发
  - ERC20智能合约开发
  - 合约API测试脚本
  - 后端服务集成

### dev/front_end
- **前端开发分支**
- **完成内容**：实验2.2 - ERC20合约前端集成
  - JYMToken合约前端部署
  - 钱包联动功能
  - 前端界面开发
  - 部署指南文档

## 实验完成情况

- ✅ **实验2.1** (dev/back_end)：后端ERC20合约开发与测试
- ✅ **实验2.2** (dev/front_end)：前端JYMToken部署与钱包集成
- ✅ **项目整合** (main)：前后端功能完整集成

## 技术栈

- **智能合约**：Solidity
- **后端**：Node.js + Web3.js
- **前端**：React + Hardhat
- **代币标准**：ERC-20
